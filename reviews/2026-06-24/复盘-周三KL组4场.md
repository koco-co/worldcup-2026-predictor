## 二、逐场归因（λ 仅用赛前信息推，不拿结果反推）

### 1. 葡萄牙 5:0 乌兹别克斯坦 —— 漏判“生死需大胜”，又被“不可零封”毙掉正解
- **根因①（头号）漏判战意**：葡萄牙 R1 仅 1 分垫底于三队之末，**不大胜则末轮打领头哥伦比亚生死、出局风险陡增**——教科书级“生死·需大胜（强打弱）”。报告只口头写“必须取胜止颓”，没把出线压力升格为**能改判比分量级**的一等因子，仍押小胜 2:1。
- **根因②“碾压不收敛”没执行**：把最弱队（0 分、上限低）当“守得住、破门耗时”，默认收敛到净胜 1–2，忽略弱旅在生死压力 + 被持续投入下**崩盘成漏勺、净胜远超 1 球**。
- **根因③ 既有规则帮倒忙**：λ_客 0.75“接近 0.8”+ 硬判 BTTS“是”→ 触发“首选不可零封”，把 2:0/3:0/4:0/5:0 全从首选剔除；“判 C 必含 BTTS 比分”进一步锁死零封大胜这一真实剧本。**实际葡萄牙正是零封 5:0。**
- **λ：old [2.1, 0.75] → new [2.9, 0.5]**（葡需大胜 + 对手最弱 + 室内利技术流 → 主队上调；客队反击预期是对哥伦比亚的样本，面对必须压上的葡应降）。

### 2. 英格兰 0:0 加纳 —— 对手 C 判成 B（应然当实然）+ 低估平 + GIGO 放大
- **根因① 对手判型反了**：加纳首轮是“60 分钟龟缩 + 95 分钟绝杀弱旅”=**B 纪律铁桶**本色。报告据 ESPN“加纳**应**主动上抢”+ 几名攻击手判成 **C 开放对攻**——**把“教练应该怎么打”（应然）当成“球队实际会怎么打”（实然）**。弱队对强队、平局可接受时，务实主帅（Queiroz）多半龟缩。
- **根因② 低估平 + 漏判战意**：英、加各 3 分形势大好、平局对双方都不亏，报告口头承认却仍押英直接胜 70%、把平压到 17% 脚注。
- **根因③ GIGO 放大**：判型一错，“C→总进球强制升档到 3”+“λ_客 0.88≥0.8→不可零封”联手把 0:0/1:0 等**唯一正解主动禁掉**，复式 5 注全是“英格兰进球”路径。
- **λ：old [2.4, 0.88] → new [1.55, 0.65]**（撤 C 升档虚高 + 落实控分收力 → 主队降；判对 B + 失核心 Kudus + 平可接受 → 客队降到 0.7 以下、不再触发禁零封）。新 λ 下平局 25%（从 17% 提为 co-main）。

### 3. 巴拿马 0:1 克罗地亚 —— λ 先高估、再被“不可零封”踢掉矩阵第一
- **根因① λ_巴 0.85 偏高是源头**：巴拿马队史世界杯 4 战仅 2 球、本场又是双方都怕先丢球的低烈度生死战，应在 0.55–0.65，却卡在 0.8 阈值线上。
- **根因② “不可零封”踢掉了矩阵本就第一的 0:1**：即便用旧 λ(0.85/1.75) 复算，**0:1 也是单一最可能比分（12%）**，是硬约束把它从首选改成了 1:2。规则帮倒忙。
- **根因③ 认知-数字不一致**：正文反复写“怕先丢球/低烈度/平被低估”，λ 却给到总进球 2.6——叙事识别对，数字没压下来。
- **λ：old [0.85, 1.75] → new [0.6, 1.2]**（双方相互忌惮 → 双降，总进球回到 1.8）。

### 4. 哥伦比亚 1:0 刚果(金) —— 领头羊求稳，λ_刚高估到 0.80 又踩零封禁令
- **根因① 漏判“领头羊求稳锁分”**：哥已 3 分领头、平即基本出线、为末轮战葡留力 → 求稳控速。**求稳强队打不开铁桶的典型结局就是 1:0，不是 2:1。**
- **根因② λ_刚 0.80 高估**：刚果对葡的唯一进球来自定位球后点（Masuaku→Wissa），面对的是必须压上的更强葡；面对求稳、边卫不压上的哥伦比亚，其得分路径被掐断，合理 λ≈0.55–0.65。0.80 恰好踩线触发“不可零封”，把**模型自己的第一档 1:0** 改押 2:1。
- **根因③ 自检被规则绑架**：自检里白纸黑字写了“1:0 为模型最高档但零封、降级处理”——**看见正解却为合规主动放弃**。
- **λ：old [1.65, 0.8] → new [1.3, 0.6]**（求稳压净胜 + 刚果得分路径被掐 → 双降，1:0 成主导档 19%）。

## 三、系统性偏差（跨场）

1. **战意/出线情景没升格为一等因子（4 场全中招）**：葡（生死需大胜）、英加（双方求稳/避险）、哥（领头羊求稳）、巴克（双生死相互忌惮）——**全部是出线情景驱动的剧本**，报告都口头识别了、却只当总进球微调，没让它改判比分量级与胜负倾向。这是本届反复出现的同一个病。
2. **“λ≥0.8 / BTTS→不可零封”+“C→强制升档必含 BTTS”被批量过度套用（本批最致命）**：4 场实际是 5:0、0:0、0:1、1:0——**3 场零封、总进球都低**。这两条规则建立在“对手会进球”的乐观前提上，把真实的零封/低分剧本系统性禁掉。**dc 回测显示巴 0:1、哥 1:0 在旧模型里本就是 rank 2 的高概率比分，是规则把它们从首选踢掉的。**
3. **把“应然”当“实然”判对手类型**：加纳因“媒体说该上抢”被判 C，实为 B 龟缩。判型一错，下游升档/禁零封跟着错（GIGO）。
4. **认知-数字脱节**：报告正文识别了“低烈度/求稳/怕丢球”，但 λ 估值没跟着压下来——“嘴上承认、数字忽略”。

## 四、对抗式验证（过拟合检查，逐条自我挑刺）

> 综合与对抗两步原计划由子智能体跑，因 session 限额失败，由主分析师补做并刻意挑刺。

- **英格兰 0:0 真的是“控分避强”吗？** —— **部分是，别硬塞。** 英格兰其实尽力了（控球占优、凯恩有机会），主因是**加纳 B 铁桶被判成 C + 散户低估平**，“双方求稳”只是助推。**结论：不把英格兰这场归为纯控分**，主修是“判 C 需实然证据”+“别低估平”，控分为辅。避免把每个失误都套进“控分”。
- **“零封刹车”会不会误伤过去该升档的场（荷兰 5-1、挪威 3-2）？** —— **不会，因为豁免是有条件的。** 荷兰-瑞典、挪威-塞内加尔有**实然开放对攻数据**（近期真对攻、零封率低）→ 该升档；本批是**战意驱动的低烈度/断层碾压**、对手并无实然反击 → 该刹车。新加的“**判 C 必须实然证据**”正是这条边界：实然对攻→升档；应然/动机驱动→刹车。
- **new λ 是不是拿 5:0/0:0 结果反推的事后诸葛？** —— **大体不是。** λ 理由都引赛前信息（Opta 胜率、欧赔、火力史、Kudus 缺阵、积分形势）；且 dc 显示**巴 0:1、哥 1:0 在 OLD 里已是 rank 2**（NEW 只是去掉人为虚高、解除禁令）。诚实保留：**葡 5:0（NEW 仍 rank 7）、英 0:0（NEW rank 4）即便修正后仍是尾部**——新框架能把**方向 + 量级 + 总进球档 + 敢押零封**救对，**精确比分仍难 100% 命中**。
- **3/4 方向对却收 0，问题在判型还是玩法？** —— **两者都有，但玩法是放大器。** 即便判型完美，**4 腿全精确比分串关本身就是彩票级**；本批 4 场又都是“强队 + 精确比分”高相关、all-or-nothing。**真正的杠杆是：当方向置信 ≫ 比分置信时，改用让球/双重机会/总进球，别堆精确比分金字塔。**

## 七、配资 / 串关教训

- **4 场全押精确比分金字塔 = 高相关 all-or-nothing**：3/4 方向对仍收 0。精确比分串关命中难、错一场全层灭。
