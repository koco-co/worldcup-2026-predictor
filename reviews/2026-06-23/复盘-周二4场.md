## 二、逐场复盘

### 1. 法国 3-0 伊拉克 —— ✅ 满分（首选完全命中）

- **预测**：3:0 主胜、总进球 3（偏 4）、伊拉克＝A 崩盘漏勺。
- **实际**：3-0，姆巴佩 2 球、登贝莱 1 球；费城雷暴致比赛延迟。
- **复盘**：A 漏勺判断＋「碾压场比分别收敛到净胜 1」执行到位，直接给 3:0 而非 2:1。**天气因素也判对**——预判费城雷暴会压低节奏，故没冒进押 4:0/5:0，最终 3:0 刚好。这是本批教科书级的一场。

### 2. 阿根廷 2-0 奥地利 —— ✅ 比分全中（总进球押高 1 球）

- **预测**：2:0 主胜、总进球 3、奥地利＝C 开放对攻（BTTS 偏是）。
- **实际**：2-0，梅西 2 球（生涯 18 球，超克洛泽成世界杯历史第一射手）；奥地利被零封。
- **复盘**：比分主判精准。唯一小偏差——判 C 对攻「BTTS 偏是」，实际奥地利没进球（被零封），总进球押 3 实际 2。**奥地利虽是攻击型阵型，但碰阿根廷被压制后并未打出对攻**，更接近「被零封的强弱分明」而非真对攻。提示：C 类里要再细分「自己也强、真能对攻」与「纸面攻击型、遇强队仍被零封」。

### 3. 挪威 3-2 塞内加尔 —— ⚠️ 胜负对、比分/总进球低估（**核心失分点**）

- **预测**：2:1 主胜、总进球 3、塞内加尔＝C 开放对攻、BTTS 是。
- **实际**：3-2。Pedersen（库利巴利失误送礼）1-0 → 哈兰德 2-0 → Sarr 2-1 → 哈兰德 3-1 → Sarr 补时 3-2。**双方对攻、5 球、塞内加尔进 2**。
- **复盘**：**方向全对**（C 对攻、BTTS 是、挪威胜、净胜 1）——但**进球量级整体低估一档**：押总进球 3 实际 5、押 2:1 实际 3:2。**这正是「判了 C 开放对攻却没把总进球升够档」的老毛病**（SKILL.md 2026-06-21 复盘已写过「λ 全挤在 2.69–3.31」，本批 4 场首选总进球又挤在 2–3 档）。C 类的本质是「高总进球＋高方差」，比分却给了保守的 3 球。

### 4. 约旦 1-2 阿尔及利亚 —— ⚠️ 胜负对、比分押零封漏球（**自洽性失分点**）

- **预测**：客胜、首选 0:1（用户版）/ 0:2（原版）、总进球 2、约旦＝B 铁桶带 A 漏勺渗漏。
- **实际**：约旦 36' 先进球（Alrashdan）1-0 → 阿尔及利亚 69' 头球扳平（马赫雷斯角球助攻，Benbouali）→ 82' Gouiri 逆转，2-1。**约旦先领先、还进了球**。
- **复盘**：**胜负判对很难得**（主队先进球的逆境下仍守住客胜判断，对手类型「带漏会被逆转」也对）。但**比分押 0:2/0:1 零封是错的**——分析里明明写了「约旦带 A 漏勺渗漏、保留 BTTS、约旦会进球」，**最终比分却押了约旦零封**，自相矛盾。且阿尔及利亚破桶靠**定位球（角球→头球）**，印证「破密防靠定位球」。

## 四、系统性偏差（本批根因）

做对的（保持）：

1. **胜负倾向 4/4**——胜负判断已是稳定强项，尤其顶住了约旦场「主队先进球」的冷门压力。
2. **A 漏勺/碾压场比分精准**——法国 3:0、阿根廷 2:0，对手类型 A + 「碾压别收敛净胜 1」执行到位。
3. **天气/节奏纳入**——费城雷暴压低法国总进球判断正确。

要修的（两条核心偏差，且都是「分析里已识别、最终却没落到比分」的脱节）：

| 偏差 | 表现 | 根因 | 反例场 |
| ---- | ---- | ---- | ---- |
| **C 开放对攻场总进球低估** | 判了 C 类，比分/总进球却仍给 2–3 档保守值 | 「C 类＝高总进球高方差」只写在原则里，没有可执行的「升档」动作 | 挪威 3-2（押总进球 3，实际 5） |
| **对手会进球却押零封** | 写了「对手带漏/BTTS 偏是」，首选比分却押零封 | 自洽性自检没把「BTTS 判断 ↔ 比分是否零封」列为硬约束 | 约旦 1-2（押 0:2/0:1，约旦进 1 还先领先） |

> 一句话：本批的短板**不在胜负、不在方向，全在「比分量级」**——尤其 C 开放对攻场总进球该升档没升、对手判定会进球却押了零封。这两条恰好也是输掉中层/封顶层的直接原因。

## 五、策略优化建议（待固化进 skill）

1. **C 开放对攻场比分强制升档**：一旦判定对手为 C 类（或本场判为开放对攻），首选总进球档在常规判断上**+1 档**，且比分复式**必须含「双方都进」的 BTTS 比分**、不得只押零封小胜。
2. **新增自洽性硬约束「BTTS ↔ 零封」**：若分析中写了「对手会进球 / 带漏勺渗漏 / BTTS 偏是」，则**首选比分不得为零封**（X:0 / 0:X）——出现即视为自相矛盾，回改。
3. **延续复盘惯例**：把本场（2026-06-23）案例追加进 SKILL.md 的复盘记录，与 2026-06-21/06-22 并列，作为「胜负对但比分量级低估」的标本。
