## 模块二　关键对位与战术博弈

### 核心对位

1. **Hakimi（摩右路）⚔️ 海地左路**：Hakimi 持续前插下底是摩洛哥最稳定的边路爆点之一，对位海地脆弱边后卫，是**净胜球向上的主要通道**（[Rotowire](https://www.rotowire.com/soccer/article/morocco-vs-haiti-preview-predicted-lineups-team-news-tactical-analysis-2026-world-cup-group-c-119340)）。
2. **Diaz / Ounahi（摩中前场创造力）⚔️ 海地中卫线**：海地后防 0-3 被巴西**打穿**，面对摩洛哥多点创造力**很难全场守住**，摩洛哥多人具备终结/做球能力，进攻点分散难以盯防。
3. **海地双前锋 Isidor / Pierrot（+ 存疑的 Nazon）⚔️ 摩洛哥中卫 + Bounou**：海地"放开踢"的唯一指望，但摩洛哥防守扎实、Bounou 稳健，海地 2 战零进球，**这条线偷一个的概率是尾部事件、非主线**。

### 中场控制 / 定位球

- 中场控制权**摩洛哥几乎全程占优**（El Aynaoui + Bouaddi 控制 + Ounahi 串联），海地难以组织有效推进。
- 定位球是摩洛哥额外得分通道之一（身体与落点优势），对海地脆弱后防可形成补充威胁；但**无可靠定位球转化样本，不夸大**。

### 双方弱点与突破口

- **海地弱点**：后防被巴西打穿、整体班底差距巨大、已出局战意松散、唯一射手 Nazon 存疑 → 摩洛哥**多点持续施压**即可建立优势。
- **摩洛哥弱点**：**本届终结效率一般**（2 战仅 2 球），有"碾压控球但临门一脚浪费"的风险；提前确认出线后**可能下半场降速轮休**，压缩净胜球 → 这是"赢但赢得少"（1:0 / 2:0 而非大胜）的直接来源。

### 对手类型判定：海地＝**A 类（崩盘漏勺）**

- **判 A 的真实依据（均已联网核实）**：① 海地 2 战实然**丢 4**、**0-3 被巴西打穿后防**；② **已出局**、纯荣誉战，"放开踢"只会更暴露后防；③ 海地无进球（2 战 0 球），不存在"开放对攻能反咬"的实然依据。对手＝A 漏勺 → **净胜别不默认收敛、复式向上覆盖**（碾压场可多打几个）。
- **摩洛哥侧自评＝C 类风格队**（有控球、有效率、防守扎实的开放对攻方）面对 A 漏勺 → 属**实力悬殊碾压 + A 漏勺**组合：摩洛哥可持续多点施压，净胜空间向上而非向下。
- **零封路线成立**：海地 λ 0.50 < 0.8、A 漏勺、2 战 0 球 → **碾压场可取零封首选（2:0）**，且按 A 漏勺"净胜别不默认收敛"向上盖 3:0；**战意＝领头求胜优先于 BTTS 零封禁令**，零封首选合规（详见模块四自洽性自检）。

### 教练博弈

- 摩洛哥主帅：已出线但要争第 1，会**先排强阵求多进球**，待出线与净胜差明朗后**才考虑保护性轮休/降速**——这意味着上半场强度高、下半场可能松。
- 海地主帅：已出局，倾向"放开踢"博荣誉、双前锋出击，但缺乏可执行的进攻火力 → 实然效果是**防线被进一步拉开**，而非真正威胁摩洛哥球门。
