# 竞彩 / 体彩足球玩法规则参考

涉及**竞彩具体玩法**（让球如何结算、半全场有哪些组合、混合过关怎么算、各盘口怎么区分）时读取本文件，以保证口径一致、不靠临场记忆。

「体彩竞彩」即中国体育彩票旗下的**竞彩足球**。竞彩采用**固定赔率（SP）**，奖金 = 投注本金 × 赔率。

> **结算时间口径（很关键）：** 竞彩各玩法除特别说明外，均按**90 分钟（含伤停补时）**结算，**不含加时与点球**。淘汰赛即使打加时/点球决出晋级，竞彩仍按 90 分钟结果算盘——分析淘汰赛盘口时务必记住这条。

---

## 一、胜平负（含让球）

竞彩的胜平负分两种售卖：

| 玩法 | 说明 | 结算 |
| ---- | ---- | ---- |
| 不让球胜平负 | 直接选主胜 / 平 / 客胜 | 按 90 分钟实际比分 |
| 让球胜平负 | 官方给一个**整数**让球（如 -1、+1、-2） | 用「实际比分 ± 让球数」判定胜 / 平 / 负 |

**让球结算逻辑（与 SKILL.md 原则 3 一致）：** 以主队让一球（-1）为例——

| 实际结果 | 让球后 | 竞彩判定 |
| ---- | ---- | ---- |
| 主队净胜 2 球及以上 | 仍为胜 | **让胜** |
| 主队刚好净胜 1 球 | 抵消后为平 | **让平** |
| 主队平或负 | 让球后为负 | **让负** |

要点：竞彩让球是**整数**（没有 -0.25、-0.75 这类半球盘），不存在「赢半输半 / 走水」，结果只有让胜 / 让平 / 让负三种。

---

## 二、比分

在官方给定的比分网格内选具体比分（如 1:0、2:1、0:0…）。比分超出网格范围时，用兜底选项：

- **胜其他**：主队以网格外的大比分获胜
- **平其他**：平局且比分超出网格（如 4:4）
- **负其他**：客队以网格外的大比分获胜

> **比分复式选档要接对手类型（见 SKILL.md「对手类型分级」、[parlay-mode.md](parlay-mode.md)「比分复式覆盖形状」）：** 覆盖哪几档不是默认押一簇窄区间小胜——**A 漏勺/碾压场**向上含大胜尾部(净胜 3、4 球)并勾**「胜其他」**防出网格；**C 对攻/博客胜场**含「对方也进一球」的 **BTTS 比分**(2-1/3-2)；**B 铁桶场**比分难猜，首选放弃比分复式、退让球/胜负。覆盖档要跨净胜球数分散方差。

---

## 三、总进球数

双方 90 分钟内进球**合计**，按区间投注：**0、1、2、3、4、5、6、7+**（7 球及以上归一档）。

> **总进球数是竞彩的进球类玩法，不要和「大小球」混。** 中国体彩竞彩**不售大小球（over/under 2.5 这类）**——大小球是亚盘/欧赔/外围市场的玩法。所以盘口是竞彩/体彩时，进球判断走**总进球数（0–7+）**；只有用户看的是亚盘/欧赔/外围盘时才用大小球。（官方接口的进球玩法 poolCode 只有 `ttg` 总进球，没有大小球，可佐证。）

---

## 四、半全场

「半场结果 / 全场结果」的组合，共 **9 种**（每段均为胜 / 平 / 负）：

| | 全场胜 | 全场平 | 全场负 |
| ---- | ---- | ---- | ---- |
| **半场胜** | 胜/胜 | 胜/平 | 胜/负 |
| **半场平** | 平/胜 | 平/平 | 平/负 |
| **半场负** | 负/胜 | 负/平 | 负/负 |

半全场波动大，通常只适合小注 / 进取方案（见 SKILL.md 原则 5）。

## 五、单关 / 过关（串关）/ 复式 与奖金计算

### 单式 vs 复式（多选，中一即中）

- **单式**：每个玩法只选 **1 个**结果（如只押「让胜」）。
- **复式（多选）**：同一玩法里**勾选多个结果**（如比分同时押 2:0 / 2:1 / 1:0，总进球同时押 2 / 3 档）。
  - **注数 = 该玩法的选择数**；单场复式投注额 = 单注额 × 注数。
  - 实际结果命中**任一**被选项即中奖，奖金按**命中那一项**的赔率算。
  - 所以「比分」绝不是只能押一个——**比分、总进球、半全场都能复式覆盖多个结果**来提高命中率，代价是注数 / 成本上升。

### 单关 / 过关（串关）

- **单关**：**部分玩法 / 场次**开放单场投注（只买一场也算一注）；哪些玩法、哪些场次可单关以官方实时标识为准（接口 `poolList[].single`）。
- **过关（串关）**：把**多场**比赛组合成一注，如 2 串 1、3 串 1 …… N 串 M（**跨场**组合，同一场不同玩法不能互串）。
  - **任一场不中，整串作废**（M 串 1 需全部命中才派奖）。
  - **复式串关**：每条腿也能复式多选，总**注数 = 各腿选择数之积**。
- ⚠️ **会变动的规则（诚实标注）：** 可单关 / 可串场次、过关方式（串关张数）、投注与截止时间等，**以中国体育彩票官方公布为准，且随场次和政策调整**。不要把过期限制当确定规则；用户问具体能否串、能串几场，提示其以官方实时为准。

### 投注额与奖金计算（竞彩离散定额口径 · 唯一权威定义）

> **铁律：竞彩每注固定 2 元，投入必须正向乘出，禁止用「总额 ÷ 注数」反推单注。**
> - **总投入 = 2 元 × 注数 × 倍数**（倍数为**正整数**，不加倍即 1 倍）。
> - **注数**：单式 = 1；单场复式 = 选择数 k；复式串关 = 各腿选择数之积 ∏(选择数)。
> - 任何投入都必须是 **2 × 注数** 的整数倍。例：9 注复式合法投入只能是 18、36、54……（18 的整数倍），**不可能是 40 元**（40 ÷ 9 = 4.44 元/注是竞彩里不存在的非法单注）。

**标准算法（先定倍数，再乘出投入与奖金）：** ① 定注数 → ② 定倍数（1 注本 = 2×注数；配资见下「占比→合法倍数吸附」）→ ③ 投入 = 2 × 注数 × 倍数 → ④ 理论最高奖金 = **2 × 倍数 × 命中档（连乘）最高赔率**，与注数无关。

| 形态 | 注数 | 1 注本（=2×注数） | 投入 | 理论最高奖金（上限） |
| ---- | ---- | ---- | ---- | ---- |
| 单场单式 | 1 | 2 | 2 × 倍数 | 2 × 倍数 × 赔率 |
| 单场复式（多选 k 项） | k | 2k | 2k × 倍数 | 2 × 倍数 × **max(各选项赔率)** |
| M 串 1 单式 | 1 | 2 | 2 × 倍数 | 2 × 倍数 × **∏(各场赔率)** |
| 复式串关 | ∏(选择数) | 2 × ∏选择数 | 2 × ∏选择数 × 倍数 | 2 × 倍数 × **∏(各腿最高赔率)** |

- 「理论最高奖金」是**命中覆盖结果中赔率最高那一注**时的上限，命中其它被选项更低——报告写明「上限值、非保证」；注数只决定**成本与覆盖面**，**不进单注奖金分母**。
- **举例（复式串）**：场1 比分 3 选 × 场2 比分 3 选 → 注数 = 9，1 注本 = 18 元；投 2 倍 → 投入 = **36 元**（只能是 18 的整数倍，不能写 40 元）；若命中那注两腿 @8.0、@7.5，理论最高奖金 = 2 × 2 × 8.0 × 7.5 = **240 元**。
- **占比→合法倍数吸附（配资用）**：50/40/10 是**目标占比**、不是能精确落地的金额。落地：① 1 注本 = 2 × 注数；② 倍数 = round(预算 × 占比 ÷ 1 注本)，至少 1；③ 实际投入 = 2 × 注数 × 倍数。吸附后各卡总额与预算常有小出入，**报告须写明「占比为近似、实际投入为合法 2 元倍数」，不得出现 4.44 元这类非法单注**。
- 算「理论最高奖金」用的赔率尽量取**官方实时赔率**（见 [data-sources.md](data-sources.md) 竞彩官方源），取不到时标为估计值。

## 六、四种盘口「钉死区分」

分析让球 / 赔率时，**先判断属于哪一类**，禁止混用（对应 SKILL.md 原则 3）：

| 盘口 | 让球形式 | 赔率 / 水位 | 判定关键 |
| ---- | ---- | ---- | ---- |
| **竞彩让球** | **整数**让球（-1、+1、-2…） | 固定赔率 SP | 实际比分 ± 整数 → 让胜/让平/让负，无走水 |
| **亚洲盘口** | 可含**半球 / 球半 / 四分之一球**（-0.25、-0.75、-1.25…） | 水位 + 升降盘 | 有赢半输半、走水；要看水位与升降方向 |
| **欧洲赔率** | 无让球 | 主胜 / 平 / 客胜固定赔率 | 判市场胜负倾向与冷门防范 |
| **大小球** | 总进球线（2.5、3、3.5…） | 水位 | 判进球预期，看大/小球降水。**属亚盘/欧赔/外围，竞彩不售** |

典型易错点：① 把「竞彩让 1」（整数）和「亚盘 -1.5」（球半）当成同一回事——两者结算逻辑完全不同；② 给一场**竞彩**比赛硬套「大2.5/小2.5」——竞彩没有大小球，进球判断应走总进球数（见第三节）。必须分开判断。
