## 五、押注方案与推荐（默认体彩竞彩模式）

基于前四大模块（已过结论自洽性自检）的判断给出押注方案。**默认盘口＝体彩世界杯竞彩模式**——竞彩五大玩法：胜平负 / 让球胜平负 / 比分 / 总进球数 / 半全场胜平负（**无大小球**，进球类只用总进球数）。规则见 SKILL.md 第三步。

至少给 2–3 套、覆盖不同风险档；对最优一套在「方案」名后标「**推荐**」。**方案里的「参考赔率」尽量取自官方接口的真实赔率**（`had`/`hhad`/`crs`/`ttg`/`hafu`，取数见 [data-sources.md](data-sources.md)）；取不到则标注为估计值。方案可**复式覆盖多个结果**（比分 / 总进球同时押几档，中一即中）；每套算出「**理论最高奖金**」（命中最优时的上限，公式见 [betting-rules.md](betting-rules.md) 第五节）。

| 方案 | 风险档 | 玩法 | 覆盖选择（可复式） | 参考赔率 | 投入 | 理论最高奖金 | 理由 |

| 方案 C | 以小搏大 | 比分 / 半全场 / 总进球高档 | 【比分复式按对手类型选形状：A 漏勺/碾压含大胜尾部 3:0/4:0+胜其他；C 对攻/博客胜含 BTTS 2:1/3:2；B 铁桶不建议押比分。详见 parlay-mode.md】 | 【】 | 【】 | 【单注×最高赔率】 | 【博精确 / 博冷】 |

**配资：** 用户给押注金额 N 元就用 N，**未给则默认以约 128 元为基数**（用户惯常区间 100–150、典型 72+32+20≈128，注明"默认基数，按自身预算调整"）。在方案间分配，稳健占大头、搏大只用小注——稳健 X 元 / 中庸 Y 元 / 搏大 Z 元，合计 = 本金；每套列出**理论最高奖金**（复式 / 串关按 [betting-rules.md](betting-rules.md) 第五节算）。
**止损纪律：** 单场总投入 ≤ 愿亏额度（默认基数约 128 元、上限按用户区间 150 封顶）；以小搏大 ≤ 总额 20%；不追高、不回本式加注。串关（需跨场）按 [betting-rules.md](betting-rules.md) 第五节提示「任一场不中整串作废、可串场次以官方为准」，命中率随场次下降，更要控小注。

**推荐说明：** 推荐【方案 A】，因为【一句话：更稳 / 性价比更高】。「推荐」是相对更优，**不等于稳赢**。

**风险提示：** 【最大不确定性、以小搏大方案的愿亏额度、不要追高】。

## 可选追加需求

用户提出更深需求时，按对应模块追加分析。

- **追加盘口分析：** 额外分析亚盘初盘、即时盘、欧赔变化和大小球盘口走势，判断市场倾向与潜在风险。
- **追加竞彩让球分析：** 客户看到的是竞彩让球盘，按让胜、让平、让负逻辑分析，并给出对应比分。
- **混合方案：** 已作为核心步骤，见上文「五、押注方案与推荐」（默认体彩竞彩模式，区分稳健/中庸/以小搏大并标推荐）。用户若要更多组合，在该节继续扩展即可。
- **追加数据模型分析：** 基于双方近期进球、失球、xG、xGA、主客场表现，建立一个简化预测模型，给出主胜、平局、客胜的大致概率区间。
- **追加阵容推演：** 基于双方伤停和近期首发情况，推演双方最可能首发阵容，并说明每个位置的战术作用。
- **追加冷门风险分析：** 重点分析本场是否存在冷门风险，包括战意、轮换、赛程、盘口异常、伤停、心理因素和市场热度。
- **追加角球与牌数分析：** 额外分析本场角球数和黄牌数趋势，包括双方边路进攻频率、压迫强度、裁判尺度和比赛对抗强度。
