# 串关综合报告 · 6/24 凌晨四场（K组/L组）

> 配资基数 100 元演示。**本批经【批次相关性闸门】判为「高相关批次」，已主动收缩至总投入 80 元**（详见下文）。竞彩每注 2 元，投入 = 2 × 注数 × 倍数；结算按 90 分钟（含伤停，不含加时点球）。赔率取报告内复式覆盖值，部分为估值，**以官方实时 SP 与 single 标识为准**。

## 一、批次相关性闸门（横向聚合 → 判定：高相关）

| 场次 | 对手类型 | 胜负置信 | 风险栏关键判定 | 可入保底层？ |
| ---- | ---- | ---- | ---- | ---- |
| 葡萄牙 vs 乌兹别克斯坦 | **C 开放对攻** | 主胜·中 | 非真实平局、非铁桶降级，让胜可作核心腿；但葡终结低迷可能净胜收窄 | ✅ 唯一合格 |
| 英格兰 vs 加纳 | **B 铁桶** | 主胜·中 | 真实平局中等 + 加纳触发铁桶降级 + 英格兰直接胜置信仅中——**明令不得作保底核心腿** | ❌ |
| 巴拿马 vs 克罗地亚 | **B 铁桶** | 平局·中 | 真实平局风险**高**、置信仅中——**不得作保底核心腿** | ❌ |
| 哥伦比亚 vs 刚果金 | **B 铁桶** | 主胜·中低 | 真实冷平存在 + B 铁桶触发铁桶降级 + 置信中低——**直接胜腿不作保底核心腿** | ❌ |

**判定结论：高相关。** 四场里有**三场同为 B 纪律铁桶**、且**三场都显式标注「真实平局/冷平 + 铁桶降级 + 置信仅中或中低」**——这就是同一个系统性冷门源（密防摆铁桶 + 强队破密防被高估 + 真平被低估），方向一错比分必错、会一起翻车。远超「≥2 场共享同一冷门源」的高相关门槛。按【高相关批次收缩规则】，**不准默认 3 卡满铺**。

**收缩动作（写明砍了哪些同坑场）：**
1. **砍场次**：三场 B 铁桶同坑，**保底层只保留 1 场非同坑场**（葡萄牙 C 对攻），其余两场（英格兰、克罗地亚/巴拿马、哥伦比亚）不进保底层。
2. **降串数**：保底层由「2 串 1」**降级为单场比分复式**——因为合格的可读场只剩葡萄牙一场，凑不出第二条干净的保底腿，不硬塞 B 铁桶进去假装稳。
3. **先砍封顶 + 中层**：中层 3 串 1 倍数压到 1、封顶 4 串 1 倍数压到 4，仓位收回保底层。
4. **降总投入**：由 100 元基数下调至 **80 元**，省下的 20 元留作其它低相关日。

## 二、保底层选场闸门（结果：仅葡萄牙过关）

- 英格兰 vs 加纳 → 命中「B 铁桶 + 真实平局 + 风险栏明令禁入」**三项**，出局。
- 巴拿马 vs 克罗地亚 → 命中「B 铁桶 + 真实平局风险高 + 明令禁入」，出局；只可走受让/双重机会或退出。
- 哥伦比亚 vs 刚果金 → 命中「B 铁桶降级 + 冷平 + 直接胜腿禁入」，出局。
- 葡萄牙 vs 乌兹别克斯坦 → C 开放对攻、非铁桶降级、风险栏明示「让胜可作核心腿」，**唯一合格**。

→ 保底层无法凑齐两条干净腿，按闸门出路②，**保底层缩为葡萄牙单场比分复式**（这是命中率相对最高的一层，宁缩不脏）。

## 三、比分金字塔（已按高相关收缩，层层包含）

### 保底层【推荐】· 单场比分复式（葡萄牙 vs 乌兹别克斯坦）
对手 C 开放对攻 → **必含 BTTS 比分**（乌兹也会进），不押零封小胜；首选 3:1 净胜 2，向上勾大胜尾 + 胜其他防出网格、跨净胜球数分散方差。

| 覆盖比分 | 赔率 | 含义 |
| ---- | ---- | ---- |
| 3:1 | 8.0 | 首选（BTTS，净胜 2） |
| 4:1 | 12.0 | 大胜尾（BTTS，净胜 3） |
| 2:1 | 7.0 | 终结低迷收窄档（BTTS，净胜 1） |
| 胜其他 | 21.0 | 网格外大胜兜底 |

- 注数 = 4；倍数 7；**投入 = 2×4×7 = 56 元**；本票最高奖金 = 2×7×21.0 = **294 元**（命中「胜其他」时；命中其它档更低）。

### 中层 · 比分复式 3 串 1（葡 × 英 × 哥，嵌套加 2 场，倍数压至 1）
保底场（葡）+ 可读性次高的两场（英格兰、哥伦比亚），各取 2 比分、含保底那档保持层层包含。**注意：英、哥两腿命中率较低（B 铁桶 + 真平/降级），本层仅作进取补充、非保底。**

| 场 | 比分 1 | 比分 2 |
| ---- | ---- | ---- |
| 葡萄牙 vs 乌兹 | 3:1 @8.0 | 4:1 @12.0 |
| 英格兰 vs 加纳 | 2:1 @6.5 | 2:0 @7.0 |
| 哥伦比亚 vs 刚果金 | 1:0 @6.5 | 2:0 @8.0 |

- 注数 = 2×2×2 = 8；倍数 1；**投入 = 2×8×1 = 16 元**；本票最高奖金 = 2×1×(12.0×7.0×8.0) = **1344 元**。

### 封顶层 · 比分单式 4 串 1（四场各 1 首选，倍数 4，愿亏彩票）

| 场 | 首选比分 | 赔率 |
| ---- | ---- | ---- |
| 葡萄牙 vs 乌兹 | 3:1 | 8.0 |
| 英格兰 vs 加纳 | 2:1 | 6.5 |
| 巴拿马 vs 克罗地亚 | 1:1 | 7.0 |
| 哥伦比亚 vs 刚果金 | 1:0 | 6.5 |

- 注数 = 1；倍数 4；**投入 = 2×1×4 = 8 元**；本票最高奖金 = 2×4×(8.0×6.5×7.0×6.5) = **18928 元**。

## 五、结构风险与最大不确定性（不绝对化）

- **三层全是「比分」类高波动玩法，整体偏进取**：比分命中率本就低（保底层单场比分复式约 25–35% 量级、中层 3 串 1 显著更低、封顶 4 串 1 ＜0.1%），任一腿方向错即整串作废。
- **本批最大不确定性 = 三场 B 铁桶同坑**：英格兰、巴拿马/克罗地亚、哥伦比亚三场均有真实平局/冷平风险且置信仅中或中低，破密防能力被低赔高估。已据此把保底层收缩到唯一干净的葡萄牙单场、把三场同坑下放到中层/封顶愿亏区——但中层/封顶仍承担这三场的方向风险，**视作彩票级，愿亏小注**。
- **葡萄牙单场也非稳赢**：葡终结低迷，可能由 3:1 收窄到 2:1（已覆盖）甚至冷平；首发约开赛前 1 小时出，大赛轮换风险存在，**比分/盘口待首发确认，轮换风险=中**。
- **求稳者**：应进一步加大保底层（葡单场比分复式或退回葡单场让胜/总进球玩法）、砍掉中层与封顶层；或仅打保底层单卡。
- 总投入须 ≤ 自身愿亏额度，不追高、不回本式加注。**无任何一注是「必胜/稳赢」。**
